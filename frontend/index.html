Virtual Yield Frontend



This dapp uses a function that approximates 1.01^n to add 1% to each user's staked balance for each event that occurs by users who have unstaked balance. Tokens can be minted using the button at the bottom for testing purposes.

Please note, currently the rules are set such that the multiplier only increases on token mints, for testing purposes only. In future work the rules that cause the multiplier to increase could be generalized much further. Additionally to illustrate the effects better the deployed contract uses a more heavily curved function that grows more rapidly
